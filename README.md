# TeachSim-26summer

26 暑期软件工程实践项目 — TeachSim  
配有虚拟学生的 AI 辅助授课训练平台。

## 仓库结构

当前以仓库根目录作为统一开发主线，后续功能在此持续迭代：

| 目录 | 说明 | 状态 |
|------|------|------|
| `frontend/` | Vue3 前端（登录 / 课前配置 / 访谈 / 课堂 / 报告 / 历史与能力画像） | 已迁至根目录，后续前端改动请在此提交 |
| `backend/` | 后端服务 | 待对应同学迁入根目录 |
| `ai/` | AI 服务 | 待对应同学迁入根目录 |
| `docs/` | 项目文档 | 待对应同学迁入根目录 |

> `test0715/` 为前期过渡目录，其中仍保留 backend / ai / docs 等模块，请各负责人尽快迁到根目录对应位置；迁完后不再新增 `testxxxx` 目录。里程碑请使用 Git tag 标记。

## 前端快速开始

```bash
cd frontend
npm install
npm run dev
```
